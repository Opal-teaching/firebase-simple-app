ListController
controller in module firebase-example-app

Description

Controls the message list and communicates with Firebase.
Manages ./views/list.html

Dependencies

  • $scope
  • $timeout

Methods

  • clearMessages()

    Clears messages

  • initController()

    Instantiates vm.messages through a firebase events, then sets a child_added event to listen to changes.

  • pushPost()

    Pushes new message to Firebase

Events

  • $destroy

    Listens to the controller destruction and kills event listeners.
