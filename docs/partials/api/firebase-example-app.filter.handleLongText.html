handleLongText
filter in module firebase-example-app

Description

Formats text to add line breaks every lineLength number of characters.

Usage

In HTML Template Binding

{{ handleLongText_expression | handleLongText:lineLength }}

In JavaScript

$filter('handleLongText')(input, lineLength)

Parameters

ParamTypeDetails
inputstring

String to format with line breaks.

lineLengthnumber

Number of characters per new line in the formatted string.
